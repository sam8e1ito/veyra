# Storage Policy

Last updated: July 23, 2026

## Browser Storage

Veyra uses browser storage technologies to provide application functionality.

The application uses:

- Local Storage

These technologies allow Veyra to remember user preferences, authentication sessions, and application data.

---

## What Data May Be Stored

Veyra may store:

- Authentication session information
- User settings

---

## Why We Use Browser Storage

Browser storage is used to:

- Keep users signed in
- Improve application performance
- Provide a consistent user experience
- Enable Progressive Web App functionality

---

## Managing Stored Data

Users can remove stored browser data through their browser settings.

Deleting browser storage may log you out or remove locally stored application information.
