# About

**Veyra** is a mobile-first Progressive Web App that helps users stay consistent with their fitness journey.

The application allows users to:

- Create workout plans (in progress)
- Log completed workouts (in progress)
- Track meals and nutrition
- Monitor calories and macronutrients
- Set personal fitness goals
- Sync data securely with Supabase

# Getting Started

## Clone repository

```bash
git clone https://github.com/yourusername/veyra.git
```

## Install dependencies

```bash
npm install
```

## Create `.env`

```env
VITE_SUPABASE_URL=your_url
VITE_SUPABASE_ANON_KEY=your_key
```

## Start development server

```bash
npm run dev
```
